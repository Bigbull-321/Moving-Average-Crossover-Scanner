# Moving Average Crossover Scanner

A real-time stock scanner that identifies moving average crossovers in Nifty 500 stocks.

## Features
- Scans Nifty 500 stocks for MA crossovers
- Customizable moving average periods
- Bullish/Bearish crossover detection
- Interactive candlestick charts with multiple timeframes
- Real-time data from Yahoo Finance

## How to Use
1. Select short-term and long-term MA periods
2. Choose crossover type (Bullish/Bearish/Both)
3. Click "Scan for Crossovers"
4. View results and click on any stock for detailed chart
